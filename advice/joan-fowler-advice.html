Hi, my name is Joan and I have some advice for you!

Don't be afraid to make mistakes

Some of my best learning is done when i've mucked something up and i need to find a solution. It sometimes seems scary to delve into a solution without guidance but this is a necessary part of learning.

Stubbornness is a great quality for a programmer

Don't give up on a solution. It is out there somewhere and all of the time spent finding the solution is not wasted time - it results in valuable learning and development.
